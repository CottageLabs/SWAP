Create users for SWAP admin.

Create a user with your preferred username, and select if the user should have DO_ADMIN or VIEW_ADMIN privileges. Although you can tick both, a user with DO_ADMIN privileges also automatically has the ability to view admin features anyway.

VIEW_ADMIN allows a user to access the /admin area and to view any information there, but NOT to edit anything. DO_ADMIN allows a user to access the /admin area and to edit / delete / add to anything there.

Some functions, such as creating new users, or turning survey pages on and off, are only available to the super user. Super user privileges cannot be bestowed upon any new users via this interface.
